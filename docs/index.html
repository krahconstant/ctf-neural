🚀 Guide déploiement backend + clé Groq
Étape 0 — Obtenir la clé Groq GRATUITE
1. Va sur console.groq.com 2. Crée un compte (GitHub ou email) 3. API Keys → Create API Key 4. Copie la clé (commence par gsk_...) 5. Colle-la dans "Configurer API Groq" ci-dessus
Option 1 — Railway.app (gratuit · le plus simple)
1. Crée un compte sur railway.app 2. New Project → Deploy from GitHub repo 3. Sélectionne le dossier backend/ comme racine 4. Railway détecte auto le Procfile → déploie 5. Copie l'URL générée et colle-la ci-dessus
Option 2 — Render.com (gratuit)
1. render.com → New → Web Service → Connect GitHub 2. Root Directory : backend/ 3. Build Command : pip install -r requirements.txt 4. Start Command : gunicorn app:app --bind 0.0.0.0:$PORT 5. Copie l'URL et colle-la ci-dessus
Option 3 — Local (développement)
cd backend/ pip install flask flask-cors groq werkzeug gunicorn python app.py # URL Backend = http://localhost:5000
🔑 Configuration API Groq (Gratuit)
1. Va sur console.groq.com → API Keys
2. Clique Create API Key → copie-la
3. Colle-la ci-dessous — elle commence par gsk_
⚡ Groq est 100% gratuit avec un compte
